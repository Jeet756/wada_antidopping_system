# anti-doping-streamlit
# 🧪 Anti-Doping Detection Web App

This is a Streamlit-based machine learning web application that predicts the likelihood of doping in athletes based on their biological parameters.

---

## 🚀 Features

- User-friendly web interface built with Streamlit
- Predicts potential doping using a trained ML model
- Accepts input for various blood and hormone levels
- Easy to deploy and share
